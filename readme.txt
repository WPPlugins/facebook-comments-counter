== Description ==

This This plugin is addon plugin to the plugin "Facebook Comments for WordPress".

This plugin make the comments counting of the facebook's comments adds to the wordpress comments counter.

== Installation ==

1. Upload the plugin directory to the `/wp-content/plugins/` directory
1. Make sure you already install and activate ]"Facebook Comments for WordPress"](http://wordpress.org/extend/plugins/facebook-comments-for-wordpress/)
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Set the facebook application's secret under Options->Facebook Comments Counter
